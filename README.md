# 🏴‍☠️ The Thousand Sunny's Terminal

**An educational terminal-puzzle game built for the Gamedev.js Jam 2026 (Open Source Challenge).**

The Thousand Sunny has been critically damaged mid-voyage! Every ship system is throwing errors, and the crew needs your help. Take on the role of the emergency engineer, fire up the ship's terminal, and use real Linux commands to diagnose, repair, and restore the systems before the ship goes down.

## 🎮 How to Play

1. **Access the Terminal:** You'll be presented with a Linux-style terminal.
2. **Listen to the Crew:** Each system belongs to a different crew member. They will give you hints on what went wrong.
3. **Run Commands:** Use commands like `ls`, `grep`, `chmod`, `mv`, and `cat` to explore the ship's directories, modify corrupted configurations, and solve the puzzles.
4. **Repair:** Fix all core systems to win and submit your score to the global public leaderboard!
